# US Healthcare Dashboard Analysis 

Overview

Built an interactive Healthcare Analytics Dashboard using Power BI to analyze claims, policies, and premium data. The dashboard provides insights into claims performance, loss ratio, turnaround time (TAT), and renewal trends, enabling data-driven decision-making.

🎯 Business Goals

Monitor claims and loss ratio performance
Track policy renewals and premium trends
Identify processing delays (TAT)
Improve operational efficiency in claims handling

🛠️ Tools & Technologies

Power BI (DAX, Data Modeling, Visualization)
SQL (Data Extraction & Transformation)
Excel (Initial Data Handling)

📊 Dashboard Pages

🔹 Page 1: Executive Overview
Purpose: High-level performance snapshot
Includes:
Total Claims
Total Premium
Loss Ratio %
Renewal Rate %
Avg Processing Time

🔹 Page 2: Claims Analysis

Purpose: Deep dive into claims performance
Includes:
Claims by Month (Trend Analysis)
Claims by Status (Approved / Rejected / Pending)
Claims by Policy Type
High Claim Amount Distribution

🔹 Page 3: Operational Insights

Purpose: Identify inefficiencies & delays
Includes:
Average TAT by Month
SLA Performance
Delay Analysis
Bottleneck Identification

📈 Key Insights

Identified high loss ratio segments, helping reduce risk exposure
Improved reporting efficiency by 30–35% through automation
Detected delays in claim processing, enabling process improvements
Highlighted renewal trends, supporting retention strategies
